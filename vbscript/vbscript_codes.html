Electric bill calculator:
dim bill, kwh, ch ch=InputBox("What will compute?"& vbcrlf &"a. Kilowatts Per Hour (KWH)"& vbcrlf &"b. Electric Bill") function vbs() if ch="a" then bill=InputBox("How much is your bill in pesos in the last 31 Days?") kwh = bill / 10.58 MsgBox("You used about "& kwh &" Kilowatts of electricity in 31 Days") elseif ch="b" then kwh=InputBox("How much KWH did you used in the last 31 Days?") bill = kwh * 10.58 MsgBox("Your electric bill is about "& bill &" pesos in 31 Days") else MsgBox ("Invalid Input") end if end function call vbs

Pythagorean theorem calculator:
Dim c,l1,l2,h 'choices, leg 1, leg 2, and hypoteuse function vbs() c = InputBox("What would you like to solve?"& vbcrlf &"a. Hypotenuse"& vbcrlf &"b. Legs") if c = "a" then l1 = InputBox("Enter leg 1 value") l2 = InputBox("Enter leg 2 value") h = Sqr((l1 * l1) + (l2 * l2)) MsgBox("The hypotenuse is "& h) elseif c = "b" then h = InputBox("Enter Hypotenuse") l1 = InputBox("Enter one of the legs") l2 = Sqr((h * h) - (l1 * l1)) MsgBox("The missing leg is "& l2) else MsgBox("Invalid Input") end if end function call vbs
